## What is Canvloop

Canvloop is a minimal editor for creating short looping visuals for music platforms.  
Turn static images into ready-to-upload Spotify Canvas, Apple Motion covers, and social clips - without heavy software.

### Core ideas

- fast results over complex timelines  
- clean interface, zero extra buttons  
- platform-ready formats out of the box  
- works in browser

---

## Features

- image → video loop generation  
- smart 9:16 crop  
- subtle motion & filters  
- live preview  
- export mp4 for platforms

---

## Formats

- Spotify Canvas  
- Apple Music Motion  
- YouTube Shorts visuals  
- custom social clips
